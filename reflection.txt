[1] Web App นี้เป็นระบบจัดการข้อมูลลูกค้าธนาคาร สามารถแสดงรายชื่อลูกค้าทั้งหมดเรียงตาม Customer ID 
และลงทะเบียนลูกค้าใหม่โดยระบบจะสร้าง Customer ID ให้อัตโนมัติต่อจาก ID ล่าสุดในระบบ นอกจากนี้ยังสามารถ
ค้นหาลูกค้าด้วยชื่อ เบอร์โทรศัพท์ หรือ Customer ID และลบข้อมูลลูกค้าได้ครั้งละหลายรายการพร้อมกัน

[2] HTTP Method ที่ใช้ในแต่ละ function: 
- function ดึงข้อมูล (loadCustomers, searchCustomer, generateNextCustomerId) 
→ ใช้ METHOD: GET เพราะ เพราะเป็นการขอดูข้อมูลจาก database โดยไม่มีการเปลี่ยนแปลงข้อมูลใดๆ 
- function เพิ่มข้อมูล (addCustomer) → ใช้ METHOD: POST เพราะเป็นการส่งข้อมูลลูกค้าใหม่ไปสร้าง record ใน database
- function ลบข้อมูล (deleteSelected) → ใช้ METHOD: DELETE เพราะเป็นการระบุ record ที่ต้องการลบออกจาก database
